# 🌐 Language Translation Tool — CodeAlpha AI Internship (Task 1)

## 📌 Overview
This project is a **Language Translation Tool** built as part of the **CodeAlpha Artificial Intelligence Internship**. It allows users to enter text in one language and instantly get an accurate translation in another, including correct meanings for idioms, phrases, and individual words.

🔗 **Live Demo:** https://mrutunjaysenapati2005-spec.github.io/CodeAlpha_LanguageTranslationTool/

---

## 🚀 Features
- 🔤 Translate words, sentences, and idioms with accurate **contextual meaning** (not just word-for-word)
- 🌍 Supports 20+ languages including English, Hindi, Odia, Punjabi, Marathi, Gujarati, Sanskrit, Bengali, Tamil, Telugu, Malayalam, and more
- 🔁 Swap source and target languages instantly
- 🔊 Text-to-speech (Listen) for translated output
- 📋 One-tap copy to clipboard
- 📖 Shows pronunciation (transliteration) and alternate meanings for single words
- 📱 Fully responsive — works on mobile, tablet, and desktop

---

## 🛠️ Tech Stack
- **HTML5** — structure and layout
- **CSS3** — modern, responsive UI design
- **JavaScript (Vanilla JS)** — application logic, async API handling
- **Google Translate API** (via `translate.googleapis.com`) — core NLP-based translation engine
- **MyMemory Translation API** — backup translation service for reliability

---

## 🤖 Why This Counts as AI
While the interface is built with HTML/CSS/JS, the translation itself is powered by **Google's Neural Machine Translation (NMT) system** — a deep learning model trained on massive multilingual datasets. This tool integrates that AI engine into a clean, real-world usable interface, which is exactly how most production AI applications are built: a lightweight frontend powered by a robust pre-trained AI/NLP backend.

---

## ▶️ How to Use
1. Open the [live demo](https://mrutunjaysenapati2005-spec.github.io/CodeAlpha_LanguageTranslationTool/)
2. Select your **source language** and **target language**
3. Type or paste your text (word, sentence, or idiom)
4. Tap **Translate**
5. View the translation, pronunciation guide, and alternate meanings
6. Use **Copy** or **Listen** as needed
